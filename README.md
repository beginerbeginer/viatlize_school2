# Installation
1. https://github.com/beginerbeginer/vitalize_school2　
からclone（cloneしたディレクトリは覚えておくこと）
1. intellijまたはeclipseを開く
1. 上のタブ「ファイル」から開くを選択
1. vitalize_school2を選択
1. 上のタブ「実行」を選択
1. 「起動構成の実行」を選択
1. 構成の編集を選択
1. gradleを選択する
1. Gradleプロジェクトに「vitalize_school2」を入力
1. 引数にbootRunを書く
1. 「実行」を選択
